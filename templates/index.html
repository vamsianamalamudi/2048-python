About

  1. Use the arrows or W, A, S, D to move the tiles.
  2. The goal is to combine tiles with the same number to reach the 2048 tile.
  3. When two tiles with the same number touch, they merge into one and the value gets doubled.
  4. A new tile with a value of 2 or 4 will appear on the board after each move.
  5. The game is over when the board is full and there are no more valid moves.
